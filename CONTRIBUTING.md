# Contributing

Thanks for helping improve Screenlint.

## Development

```bash
npm install
npm run check
```

Please keep rules practical and deterministic. A good rule should explain what failed, where it failed, and what a developer can try next.

## 中文

感谢你帮助改进 Screenlint。

本地开发：

```bash
npm install
npm run check
```

请保持规则实用、稳定、可复现。一个好的规则应该说明哪里失败、为什么失败，以及开发者下一步可以怎么修。
