# Security

Screenlint opens pages in a local Playwright browser. Treat unknown targets the same way you would treat unknown websites.

Please report security issues through GitHub private vulnerability reporting if it is available on the repository, or contact the maintainer privately.

## 中文

Screenlint 会用本地 Playwright 浏览器打开页面。请把未知目标当作未知网站处理。

如果发现安全问题，请优先使用 GitHub private vulnerability reporting；如果仓库暂未开启该功能，请私下联系维护者。
